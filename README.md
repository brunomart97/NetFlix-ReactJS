## Como usar?
* Clone o repositório:
````
https://github.com/brunomart97/NetFlix-ReactJS.git
````
* Navegue até a pasta do projeto via terminal e execute:
````
npm install
````
* Os arquivos de dependências serão baixados!
* Após o download das dependências, execute o comando:
````
npm start
````
